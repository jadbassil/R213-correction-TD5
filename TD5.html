  • Dans le fichier backend.mjs, ajoutez la fonction qui cre un nouvel enregistrement dans votre collection :
/**
 * Ajoute un vnement  la collection "events".
 * @param {Object} data - Les donnes de l'vnement  ajouter.
 * @returns Un objet contenant le statut de l'opration et un message.
 */
export async function addEvent(data) {
    try {
        await pb.collection("events").create(data);
        return {
            success: true,
            message: "L'vnement a t ajout avec succs.",
        };
    } catch (error) {
        return {
            success: false,
            message: "Une erreur est survenue lors de l'ajout de l'vnement : " + error,
        };
    }
}
  • Dans le frontmatter de la page add.astro, traitez les donnes si la requte est de type POST et appelez la fonction addEvent pour crer un nouvel enregistrement :
let pbMessage = ""; // Initialisation de la variable pour stocker le message de retour

if (Astro.request.method === "POST") { // Vrifie si la mthode de la requte est POST
    const data = await Astro.request.formData(); // Rcupre les donnes du formulaire de la requte
    console.log(data); // Affiche les donnes du formulaire dans la console

    const eventDate = new Date(data.get("date")?.toString() || new Date()); // Convertit la date du formulaire en objet Date, ou utilise la date actuelle si non spcifie

    if (eventDate < new Date()) { // Vrifie si la date de l'vnement est dans le pass
        pbMessage = "La date doit tre dans le futur"; // Dfinit le message d'erreur si la date est dans le pass
    } else {
        const resp = await addEvent(data); // Appelle la fonction addEvent avec les donnes du formulaire et attend la rponse
        pbMessage = resp.message; // Dfinit le message de retour avec le message de la rponse
    }
}
  • Ajoutez l'affichage de pbMessage pour afficher le message de retour :
<p>{pbMessage}</p>

Dans la suite, nous allons utiliser un formulaire dans agenda/index.astro pour filtrer les vnements par catgorie.

  • Dfinissez la liste des catgories dans le frontmatter :
const categories = ['Tout', 'Thtre', 'Musique']; // Cette liste doit contenir toutes les catgories exactement comme elles sont dfinies dans PocketBase.
console.log(categories);

Vous pouvez aussi utiliser une liste dynamique :

const categories = ['Tout', ...new Set(events.map((event) => event.categorie))]; // Retourne une liste de catgories d'vnements sans doublons.

  • Dans le frontmatter, ajoutez la gestion du formulaire pour filtrer les vnements en fonction de la catgorie slectionne :
let events = await getEvents(); // Rcupre la liste des vnements
const categories = ['Tout', 'Thtre', 'Musique']; // Dfinit la liste des catgories

if (Astro.request.method === 'POST') { // Vrifie si la mthode de la requte est POST
    const data = await Astro.request.formData(); // Rcupre les donnes du formulaire de la requte

    if (categorie !== 'Tout') { // Vrifie si la catgorie slectionne n'est pas "Tout"

        categories = [categorie, ...categories.filter((c) => c !== categorie)]; // Met  jour la liste des catgories en plaant la catgorie slectionne en premier
        console.log(categories); // Affiche la liste des catgories mise  jour dans la console
    }
